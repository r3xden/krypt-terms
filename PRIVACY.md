Krypt Privacy Policy

Last Updated: June 2026

Overview

Krypt is a Discord management bot developed by Rexden under Hall Developments.

Krypt respects user privacy and only stores information necessary for bot functionality.

Information Stored

Depending on which features are enabled, Krypt may store:

- Guild IDs
- Channel IDs
- Role IDs
- User IDs
- Configuration settings
- Moderation data
- Warning records
- Ticket data
- Giveaway data
- Logging configuration
- AutoMod configuration
- AntiNuke configuration

Krypt does not collect personal information such as passwords, email addresses, payment information, or private account credentials.

Message Content

Krypt does not permanently store message content unless required by a feature such as:

- Snipe
- Ticket transcripts
- Moderation logs

Stored content is limited to functionality requested by server administrators.

Data Usage

Stored data is used exclusively for:

- Bot operation
- Feature configuration
- Moderation systems
- Security systems
- Ticket management
- Giveaway management

Krypt does not sell, rent, or share stored data with third parties.

Data Removal

Server administrators may remove configuration data by removing the bot from their server.

Certain moderation, ticket, or giveaway records may remain until manually deleted or overwritten.

Security

Reasonable efforts are made to protect stored data. However, no system can guarantee absolute security.

Changes

This Privacy Policy may be updated at any time. Continued use of Krypt constitutes acceptance of the latest version.
